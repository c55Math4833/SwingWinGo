# SwingWinGo / 搖擺視窗

**SwingWinGo** 是一個 Windows 平台的趣味工具，可以讓使用者把桌面上的視窗用「鞦韆」方式擺動（橫向、縱向、圓形、橢圓、8 字形），據說有助於緩解長時間使用電腦的視覺疲勞，並可選擇同時移動游標位置。

> 本專案為以 Go 重新實作的版本，靈感來源於 [Central Fixation](https://www.central-fixation.com/)。因為原始下載連結已無法使用，本專案以開源方式提供。
---

## 🛠 系統需求

- **作業系統**：Windows 10 / 11（已於 Windows 11 25H2 環境下完成測試）。

## 🔥 主要功能

- **視窗清單**：自動擷取並列出目前桌面上所有可見的視窗。
- **多工擺動**：支援同時讓多個視窗進行擺動。
- **多元擺動模式**：提供水平、垂直、圓形、橢圓及 8 字形等多種擺動軌跡。
- **自訂參數**：使用者可自由調整擺動的速率，以及水平/垂直方向的振幅。
- **游標連動**：可選擇開啟「當滑鼠指向目標視窗時，游標隨之連動」之進階功能。

## ⚙ 使用說明

1. 免安裝，直接下載編譯完成的 SwingWinGo.exe 檔案，放置於任意資料夾後點擊執行即可。
2. 啟動程式後，點擊 `Refresh` 以載入最新的視窗清單。
3. 於清單中勾選欲設定擺動的目標視窗。
4. 根據個人需求選擇擺動模式，並設定適當的速度與幅度。一開始建議先使用較慢的速度與較小的幅度，以免視窗移動過快或過大造成不適。
5. 點擊 `Start Swinging` 按鈕開始擺動；欲停止時，點擊 `Stop Swinging` 即可。

## 🛠 從原始碼編譯

若您希望自行編譯與開發，請參考以下步驟：

1. 下載原始碼

```powershell
git clone <repo-url>
cd <repo-folder>
```

2. 執行編譯

```powershell
go build -buildvcs=false -ldflags "-H windowsgui -s -w" -o SwingWinGo.exe .
```
